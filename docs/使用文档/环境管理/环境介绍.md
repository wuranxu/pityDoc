# 环境

  这里的环境，指的是大家对于`测试环境`, `生产环境`之间的不同定义。比如我们公司喜欢用

- DEV

  开发环境

- FAT

  开发提测环境

- UAT

  预发布环境

- PRO

  生产环境

  随着系统越来越复杂，我们可能会引入更多的`环境`。为了能够更好地区分环境，我们特此定义了一套环境的概念。

![](http://oss.pity.fun/picture/2022-2-27/1645947645793-image.png)

  配置好了环境之后，我们就可以更方便地编写`不同环境`下的`测试用例`。
